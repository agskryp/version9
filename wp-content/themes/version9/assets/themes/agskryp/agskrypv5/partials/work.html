a.g.skryp (V4)

a.g.skryp is the homepage for front-end web developer Andrew Skrypnyk.

Version 4 was built using the content management system WordPress and was developed with the purpose of learning and testing how the system works.

One of the challenges early on in developing the WordPress site was finding a free service that allowed the use of WordPress and all (or most) of it's functions. After discovering and registering with bytehost.com however, it was a simple process of setting up a mysql server, uploading the WordPress files, and installing everything on the server.

Design for the website was based largely on the how to make a WordPress website tutorial by Tyler Moore, as the main focus of this project was to learn the different tools and options WordPress has to offer.

a.g.skryp (V3)

a.g.skryp is the homepage for front-end web developer Andrew Skrypnyk.

Version 3 was designed with three main sections using the foundation framework. A home section, which introduces the user to what the website is about. A project section, cataloguing and describing the various works that have been accomplished, and an information section, giving the user additional knowledge about the web developer.

To add various flair to the website, the body background (behind the background images), and the home section sub-header text colour is animated. Clicking on the site navigation buttons will automatically scroll the page smoothly to its intended location, and projects images have a bubble effect via Bounce.js when a cursor hovers over them.

moltoDESTROYED

moltoDESTROYED is a silly web comic.

The process in building moltoDESTROYED started with designing a simple and narrow interface using the Bootstrap framework that could house the long comic strips, and be able to transition well from mobile devices to desktop computers. Large buttons were used to navigate through the comics and site pages.

The next issue was to build a solid foundation that would be simple and easy to update with each new comic. Every comic strip needed to have its own personalized link, and the navigation between each comic had to be updated and adjusted accordingly. The Angular.js framework made all of these necessities possible by wiring everything to a single JavaScript file, and the UI Router extension gave each comic its own link.

skryp6

skryp6 is the official homepage for electronic music producer Skrypnyk.

The design for skryp6.com was built using the Foundation framework, and was largely inspired by the classic Amiga computers. Additional code was added to customize the mouse cursor and window scroll-bar when using the site through Chrome or Opera to better emulate an Amiga desktop, and border animation was added to the information window to emulate a guru meditation error.

Powered using the jQuery library, the main-screen logo glitching was developed through a forked pen by Szenia Zadvornykh using TweenMax, while the news text animation was created using Typed.js.
