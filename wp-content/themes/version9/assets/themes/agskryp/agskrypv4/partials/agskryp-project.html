agskryp.com is the homepage for front-end web developer Andrew Skrypnyk.

The site was designed with three main sections using the foundation framework. A home section, which introduces the user to what the website is about. A project section, cataloging and describing the various works that had been accomplished, and an information section, giving the user additional knowledge of who he is and what he knows.

To add various flair to the website, the body background, as well as the title text colour is animated, clicking on the site navigation buttons will smoothly scroll to its intended location, and projects images have a bubble effect via Bounce.js when a cursor hovers over them.

You can view the site by visiting agskryp.com
