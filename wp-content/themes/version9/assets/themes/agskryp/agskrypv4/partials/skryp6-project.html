skryp6.com is the official homepage for electronic music producer Skrypnyk.

The design for skryp6.com was largely inspired by the classic Amiga computers using the foundation framework. Additional code was added to customize the mouse cursor and window scroll-bar when using the site through Chrome or Opera to better emulate an Amiga desktop, and border animation was added to the information window to emulate a guru meditation error.

Powered using the jQuery library, the main-screen logo glitching was developed through a forked pen by Szenia Zadvornykh using TweenMax, while the news text animation was created using Typed.js.

You can view the site by visiting skryp6.com
